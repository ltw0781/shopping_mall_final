# Spring Boot / MySQL / mybatis 를 가지고 프로젝트를 만들건데

- 로그인 정보로는 : 관리자 / 회원 / 비회원(주문과 주문정보 확인)
- 게시판 : 문의게시판 / 공지사항게시판 / 이벤트 게시판
- 주문 :  상품 페이지 / 주문확인 페이지 / 결제완료 페이지
- 유저 : 마이페이지

## 회원별 권한
- 각 권한별 설정으로는
    - 관리자 : 회원관리(회원 주문내역 취소- 회원의 요청이 있는 경우만 가능), 게시판관리(공지사항 / 쇼핑몰소식 / 이벤트), 상품(등록 / 수정 / 삭제)
    - 회원 : 상품주문, 결제, 장바구니, 마이페이지(주문정보 확인 / 회원정보 관리)
    - 비회원 : 상품주문, 주문정보로 상품 주문정보 확인
